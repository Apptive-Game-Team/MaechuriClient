## Responsive Behavior

- Modal: 90% width, max-width 1000px
- Canvas: 100% width Ã— 600px height
- Scrollable when content exceeds canvas size
- Cards: Fixed 200px width (no responsive sizing)

## States

### Default State
- Records laid out in 3-column grid
- Each card at initial position
- All cards draggable

### Dragging State
- Active card: opacity 0.5
- Cursor: grabbing
- DragOverlay: Full opacity clone
